## 프로젝트 정보


### 목적

  본 프로젝트의 목표는 등급 변경에 대한 객관적인 기준을 도입하고 게임 결과에 따라 승격 및 강등을 자동화하는 것입니다. 이를 통해 커뮤니티 구성원 간의 정서적 갈등을 줄이고 효율성을 높여 게임 커뮤니티에 활력을 불어넣고 성장을 도모하고자 합니다.
